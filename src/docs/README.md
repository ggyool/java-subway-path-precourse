## 구현할 기능 목록

### 메인 화면
- 원하는 기능을 입력받는다.
```
1. 경로 조회
Q. 종료
[예외]
- 없는 기능을 입력할 때 예
```
<br>

### 경로기준 선택 화면
- 경로를 선택하는 기준을 입력받는다.
```
1. 최단 거리
2. 최소 시간
B. 돌아가기
[예외]
- 없는 기능을 입력할 때 예
```
<br>

### 거리,시간 계산 기능
- 출발역과 도착역을 입력받아 거리와 시간을 계산한다.
```
[예외]
- 존재하지 않는 역을 입력했을 때
- 출발역과 도착역이 같을 때
- 출발역에서 도착역까지 경로가 없을 때
```

### 조회 결과 출력화면
- 결과를 출력한다.
